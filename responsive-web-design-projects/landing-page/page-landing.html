About TensorFlow

TensorFlowâ„¢ is an open source software library for high performance numerical computation. Its flexible architecture allows easy deployment of computation across a variety of platforms (CPUs, GPUs, TPUs), and from desktops to clusters of servers to mobile and edge devices.

Fast

Performance is essential when creating and deploying machine learning systems. For this reason TensorFlow includes XLA, a powerful linear algebra compiler that speeds up the execution of TensorFlow code on processors, CPUs, GPUs, TPUs, and other embedded hardware platforms.

Flexible

TensorFlow provides high-level APIs that facilitate the creation and training of your models as well as low-level control for maximum flexibility and performance.

Ready for production

TensorFlow can be used in exploratory research and even in large-scale production. You can use the same TensorFlow APIs in a variety of areas, whether creating a new model type or processing millions of requests during production.
